# Hanma CLI

The official CLI for Hanma - Grapple your backend into shape with ready-to-use snippets.

## Installation

```bash
npm install -g hanma
# or
npx hanma <command>
```

## Usage

### Initialize

Initialize Hanma in your project:

```bash
hanma init
```

This will create a `hanma.json` configuration file.

### Add a Snippet

Add a snippet to your project:

```bash
hanma add express-v5-cors
```

You can also run `hanma add` without arguments to select from a list.
